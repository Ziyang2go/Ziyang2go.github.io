Detailed Description

This class provides to ability to enumerate the render elements (see RenderElement) present in a VRay scene and to extract the data in them in different formats. Render elements are available only once the rendering of the scene has completed in production mode (NOT RT). This class is only accessible via a VRayRenderer instance.
