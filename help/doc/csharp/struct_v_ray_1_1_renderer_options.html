bool VRay.RendererOptions.AutoCommit
getset

Gets or sets a flag indicating whether changes to the currently loaded render scene should be immediately committed or, alternatively, saved with an explicit call to the VRayRenderer Commit method.

bool VRay.RendererOptions.BakeGamma
getset

Gets or sets whether gamma is baked in VRayImages. Clear (default) to keep VRayImage data in linear space. Keep in mind that VRayImage methods will apply gamma correction if necessary when you want to get bmp, jpg, png, etc. Normally should not be set.

bool VRay.RendererOptions.InProcess
getset

Gets or sets whether the executing process participates in the rendering. The default value is true. If only V-Ray slaves are rendering (i.e. the rendering is out of process), then hosts for distributed rendering should be specified with the VRayRenderer.AddHosts(String) method before starting the rendering.
